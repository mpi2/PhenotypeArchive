Highlights

Data release 3.0

In data release 3.0, we now have 1,456 mouse lines with phenotype data that has passed quality control. This is a marked increase since data release 2.0 and demonstrates the acceleration of data generation for the IMPC effort.

Statistical Analysis

The statistical analysis package used in data release 2.0 (PhenStat) has been updated to Version: 2.0.1.
